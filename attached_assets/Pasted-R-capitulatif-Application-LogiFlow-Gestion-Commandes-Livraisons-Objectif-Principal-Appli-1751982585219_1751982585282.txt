Récapitulatif Application LogiFlow- Gestion Commandes & Livraisons

🎯 Objectif Principal

Application web pour gérer les commandes et livraisons de plusieurs magasins  avec un système de calendrier centralisé et des permissions par rôle.


👥 Système d'Authentification

3 Niveaux d'Accès :


👑 Administrateur : Accès complet (tous groupes + gestion utilisateurs)

📊 Manager : Gestion fournisseurs + groupes (multi-magasins)

👤 Employé : Consultation et création dans ses groupes assignés

🗓️ Interface Calendrier Principal

Vue Mensuelle Interactive :


Navigation entre les mois

Codes couleur : 📦 Commandes (bleu), 🚚 Livraisons (vert)

États visuels : Gris = Livré, Coloré = En cours

Création rapide : Clic sur une date → Menu création


Informations Affichées :


Fournisseur (nom )

Quantité : 5P (palettes) ou 15C (colis)

Statut : ✅ pour validé



📦 Gestion des Commandes

Création :


Date, fournisseur, commentaires

Statuts : En attente → Planifié → Livré

Attribution à un groupe/magasin


Suivi :


Liaison automatique avec livraisons

Historique complet avec traçabilité

Modification/suppression (selon permissions)



🚚 Gestion des Livraisons

Création Avancée :


Quantité précise : Nombre + Type (Palettes/Colis)

Liaison optionnelle avec commandes existantes

Validation en un clic depuis le calendrier


Workflow de Validation :


Création → Statut "Planifié"

Validation → Statut "Livré" + Date automatique

Mise à jour automatique de la commande liée



👁️ Système de Détails

Modal d'Information :


Fiche complète : Date, fournisseur, quantité, commentaires

Traçabilité : Dates de création et livraison

Liens : Commandes associées avec statuts


Actions Disponibles :


✅ Validation livraison (admins/managers)

✏️ Modification complète

🗑️ Suppression sécurisée



📊 Tableau de Bord Statistiques

Métriques Principales :


Livraisons du mois en cours

Commandes en attente à traiter

Délai moyen de livraison (en jours)

Palettes & Colis livrés séparément


Analyses Détaillées :


Performance par fournisseur (graphiques en barres)

Répartition par magasin avec codes couleur

Historique des opérations



⚙️ Administration & Gestion

Gestion Fournisseurs :


Ajout/Modification : Nom, contact, téléphone

Historique des commandes par fournisseur


Gestion Groupes/Magasins :


Création avec couleurs personnalisées

Assignation d'utilisateurs multiples

Suppression sécurisée (vérification dépendances)


Gestion Utilisateurs : (Admin uniquement)


Création comptes avec rôles

Attribution groupes multiples

Gestion mots de passe

Filtrage Intelligent :
